Turn point mapper

Given a list of gliding turn points, generates a browser console command. This command generates a polygon which traces the flight path. This can be pasted into any map which uses Leaflet but is intended for use with RASP. The primary intention of this is for use when RASP is being used as a data source in gliding briefings.
